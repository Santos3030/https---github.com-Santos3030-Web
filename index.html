¿Que es HTML?

Son las siglas de HyperText Markup Language (Lenguaje de Marcado de Hipertexto) y es el lenguaje estándar que se usa para estructurar y crear el contenido de una página web. Se encarga de organizar elementos como texto, imágenes, videos y enlaces para que los navegadores puedan mostrar la información de forma coherente.



Historia HTML

Fue creado por Tim Berners-Lee en 1989 como una forma de compartir documentos científicos a través de hipervínculos. La primera versión, publicada en 1991, tenía 18 etiquetas para texto y listas. Desde entonces, el lenguaje ha evolucionado a través de varias versiones (HTML 2.0 en 1995, 3.2 en 1997 y 4.0 en 1997) que agregaron soporte para imágenes, tablas, scripts y la separación del contenido de la presentación. La versión más reciente, HTML5, lanzada en 2014, incorporó elementos para multimedia, gráficos y mejor manejo de aplicaciones web.



Editores de HTML

Son programas para crear y modificar código HTML. Hay dos tipos principales: los editores de texto (como Visual Studio Code, Sublime Text o Notepad++) que permiten editar el código directamente, y los editores WYSIWYG (What You See Is What You Get) que generan código a partir de un diseño visual (como Adobe Dreamweaver o Brackets). Las mejores opciones dependen de las necesidades del usuario, con opciones gratuitas y de pago disponibles.



Tu primer documento de HTML

Tu primer documento HTML se crea escribiendo un código básico en un editor de texto, guardándolo con la extensión .html (por ejemplo, index.html) y abriéndolo en un navegador web. La estructura fundamental incluye <!DOCTYPE html>, <html>, <head> y <body>, donde se define el título y el contenido visible de la página.



¿Que son etiquetas en HTML?

Son fragmentos de código que se utilizan para estructurar y dar formato al contenido de una página web, indicando al navegador cómo mostrar los elementos. Funcionan como marcadores que encierran el contenido, generalmente en pares de apertura y cierre, para definir títulos, párrafos, enlaces, imágenes, formularios, etc.



¿Que son atributos en HTML

Son palabras especiales dentro de la etiqueta de apertura que proporcionan información adicional sobre un elemento o modifican su comportamiento y apariencia. Se componen de un nombre y un valor y se colocan dentro de la etiqueta de apertura, después del nombre del elemento.



Reglas documentales en HTML

Con el aumento de la complejidad y la interactividad en los sitios web, es más importante que nunca adherirse a las mejores prácticas de HTML5 para ofrecer una experiencia de usuario óptima y eficiente. Veamos las reglas y normas que nos ayudan a seguir buenas prácticas en la implementación de código HTML: Estructura básica: Todo documento HTML5 debe tener una estructura básica que incluya las etiquetas <!DOCTYPE html> <html>, <head>, y <body>. La etiqueta <html> es el elemento raíz y debe contener todas las demás etiquetas. Etiquetas en mayúsculas o minúsculas: Las etiquetas HTML no distinguen entre mayúsculas y minúsculas. Por lo tanto, <HTML> y <html> son equivalentes. Se recomienda seguir siempre la misma sintaxis, preferiblemente letras en minúsculas. Caracteres especiales: Algunos caracteres especiales, como <, >, &, ", y ', tienen significados especiales en HTML. Para incluir estos caracteres en tu código, puedes usar entidades HTML (< representa el símbolo <), su código decimal (< representa el símbolo <) o hexadecimal. Ver cómo representar caracteres especiales Código con tabulados y sangrías: El código HTML indentado con tabulados facilita la lectura y su comprensión. Además, puede ayudar a evitar errores, ya que facilita la identificación de los elementos HTML. Por ello, cuando escribimos código HTML utilizamos la tabulación para entender mejor la estructura de etiquetas. Sin embargo, hay que tener en cuenta también que la velocidad de carga es uno de los factores más influyentes en la experiencia que reciben los usuarios. Por ello, para reducir el tamaño de los ficheros HTML hay que evitar los tabulados, los espacios en blanco y los comentarios innecesarios. Una vez implementado nuestro código, se puede minificar mediante herramientas específicas. Etiquetas para hacer un documento HTML

para un documento HTML incluyen <!DOCTYPE html> para definir el tipo de documento, <html> como el elemento raíz, y <body> que contiene todo el contenido visible de la página. Dentro del <body>, se usan etiquetas como <head> para metadatos, <h1> a <h6> para títulos, <p> para párrafos y <img> para insertar imágenes.
