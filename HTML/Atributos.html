Atributos globales

Son atributos que se pueden aplicar a cualquier elemento HTML, como id, class y style, aunque algunos pueden no tener efecto en ciertos elementos. Permiten añadir información adicional para identificar elementos, darles estilo, o modificar su comportamiento y accesibilidad. Ejemplos comunes incluyen id para identificadores únicos, class para agrupar elementos, style para CSS directo, lang para el idioma, y hidden para ocultar elementos.




1.1 - ID

Asigna un identificador único a un elemento, útil para seleccionar elementos con CSS o JavaScript.




1.2 - Class

Asigna una o más clases a un elemento, permitiendo aplicar estilos a múltiples elementos que comparten la misma clase.




1.3 - style

Permite aplicar estilos CSS directamente a un elemento




Atributos form

Definen su comportamiento: action especifica la URL donde se enviarán los datos, method define el método HTTP (comúnmente GET o POST), y enctype determina la codificación de los datos, especialmente para el envío de archivos. Otros atributos importantes incluyen accept-charset para la codificación de caracteres, autocomplete para habilitar o deshabilitar el autocompletado y novalidate para desactivar la validación del navegador.




2.1 - action

Indica la URL del servidor que procesará los datos del formulario.




2.2 - Method

Especifica el método HTTP (por ejemplo, GET o POST) para enviar los datos.




2.3 Atributos input

Se usa para crear controles interactivos para formularios basados en la web con el fin de recibir datos del usuario.Hay disponible una amplia variedad de tipos de datos de entrada y widgets de control, que dependen del dispositivo y el agente de usuario (user agent).El elemento <input> es uno de los más potentes y complejos en todo HTML debido a la gran cantidad de combinaciones de tipos y atributos de entrada.
